# Early Rehospitalization Forecasting in AHF

Machine Learning-based system for predicting early rehospitalization risk in Acute Heart Failure (AHF) patients.

---

## 📌 Overview

This project focuses on predicting the chances of early rehospitalization for patients suffering from Acute Heart Failure (AHF) using Machine Learning techniques. The system analyzes patient and clinical data to identify high-risk cases and support better healthcare decision-making.

---

## 🚀 Features

- Data preprocessing and cleaning
- Machine Learning prediction model
- Data visualization and analysis
- Patient risk forecasting
- Model training and evaluation
- Performance metrics analysis
